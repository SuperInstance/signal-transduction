# signal-transduction

> **How information flows through the system. From signal to response.**

[![crates.io](https://img.shields.io/crates/v/signal-transduction.svg)](https://crates.io/crates/signal-transduction)
[![license](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Signal transduction pathways for agent systems — modeling how external signals propagate through cascading reactions to produce behavioral responses. Inspired by cellular biology.

## The Biological Metaphor

In cells, signal transduction converts extracellular signals into intracellular responses through cascading pathways:
1. **Receptor** binds external signal
2. **Kinase cascade** amplifies and propagates
3. **Transcription factor** activates response
4. **Feedback loops** regulate sensitivity

This library maps the same pattern to agent systems:
- External events → receptor agents
- Information cascades → processing pipelines
- Behavioral responses → action execution
- Feedback → adaptive tuning
